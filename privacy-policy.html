Politique de confidentialité

1. Introduction

Bienvenue sur notre site web. La protection de vos données personnelles est une priorité pour nous. Cette politique de confidentialité vous informe sur les données que nous collectons, comment nous les utilisons et quels sont vos droits concernant vos données. L'utilisation de notre site web est généralement possible sans fournir de données personnelles. Lorsque des données personnelles sont collectées, elles le sont, dans la mesure du possible, sur une base volontaire.

2. Responsable du traitement des données

La responsable du traitement des données sur ce site web est notre entreprise, dont le siège se trouve en Suisse. Pour toute question concernant la protection des données, vous pouvez nous contacter à tout moment.

3. Collecte et traitement des données personnelles

Nous collectons et traitons des données personnelles uniquement dans la mesure où cela est nécessaire pour la fourniture d'un site web fonctionnel ainsi que pour nos contenus et services. Les données personnelles incluent toutes les informations qui se rapportent à une personne physique identifiée ou identifiable.

4. Données que nous collectons

a. Fichiers journaux du serveur : Lors de chaque accès à notre site web, des informations sont automatiquement collectées et transmises par votre navigateur. Ces informations comprennent :

  • Adresse IP
  • Date et heure de l'accès
  • Nom et URL du fichier récupéré
  • Le site web à partir duquel l'accès a eu lieu (URL référente)
  • Le navigateur utilisé et, le cas échéant, le système d'exploitation de votre appareil ainsi que le nom de votre fournisseur d'accès

b. Formulaire de contact : Si vous nous contactez via le formulaire de contact, vos informations du formulaire, y compris vos coordonnées, seront stockées pour traiter votre demande et, en cas de questions supplémentaires, être utilisées pour un suivi.

c. Inscription : Si vous vous inscrivez sur notre site web, nous collectons les données suivantes :

  • Nom
  • Adresse e-mail
  • Mot de passe

5. Utilisation de vos données

Nous utilisons les données collectées pour les finalités suivantes :

  • Pour la fourniture, la maintenance et l'amélioration de notre site web
  • Pour traiter vos demandes et inscriptions
  • Pour remplir nos obligations contractuelles
  • Pour analyser et améliorer l'efficacité de notre site web

6. Partage de vos données avec des tiers

La divulgation de vos données personnelles à des tiers n'a lieu que si cela est nécessaire pour l'exécution du contrat ou si vous avez donné votre consentement explicite. Toute divulgation supplémentaire ne se fera que si nous y sommes légalement obligés.

7. Utilisation de Google Ads

Notre site web utilise Google Ads, un service d'affichage de publicités. Google Ads utilise des cookies qui permettent d'analyser l'utilisation du site web. Les informations générées par ces cookies concernant votre utilisation de ce site web sont généralement transmises à un serveur de Google aux États-Unis et y sont stockées. Google utilise ces informations pour évaluer votre utilisation du site, pour compiler des rapports sur les activités du site web pour les exploitants de sites web et pour fournir d'autres services liés à l'utilisation du site et d'Internet.

8. Vos droits

Vous avez les droits suivants :

  • Obtenir des informations sur les données que nous avons stockées à votre sujet
  • Demander la correction de données personnelles incorrectes
  • Demander la suppression des données que nous avons stockées à votre sujet, à condition qu'il n'existe pas d'obligation légale de conservation
  • Demander la limitation du traitement de vos données
  • Vous opposer au traitement de vos données
  • Demander la portabilité des données

9. Durée de conservation

Nous conservons vos données personnelles uniquement aussi longtemps que cela est nécessaire pour atteindre les objectifs pour lesquels elles ont été collectées ou aussi longtemps que la loi l'exige.

10. Sécurité

Nous mettons en place des mesures de sécurité techniques et organisationnelles pour protéger les données que nous gérons contre toute manipulation accidentelle ou intentionnelle, perte, destruction ou accès non autorisé. Nos mesures de sécurité sont régulièrement mises à jour en fonction de l'évolution technologique.

11. Modifications de la politique de confidentialité

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment, avec effet pour l'avenir. La version actuelle est toujours disponible sur notre site web.

12. Contact

Si vous avez des questions concernant la protection des données, veuillez nous contacter. Nous serons heureux de vous aider.

Cette politique de confidentialité a été rédigée pour respecter les exigences légales et vous donner une vue transparente sur le traitement de vos données sur notre site web. Nous vous remercions de votre confiance et vous assurons que nous traitons vos données avec le plus grand soin et responsabilité.
